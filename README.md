# Expense Tracker Backend (Spring Boot)

A Spring Boot REST API for managing daily expenses.

## Features
- Add Expense
- Update Expense
- Delete Expense
- View Expenses
- Monthly Expense Analytics
- Category-wise Expense Analysis
- Payment Mode Analysis
- Pagination Support
- DTO Layer
- Global Exception Handling
- Swagger API Documentation
- MySQL Integration

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Swagger (OpenAPI)
